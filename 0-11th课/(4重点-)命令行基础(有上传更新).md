## echo
```
echo "hello1" > 1.txt
```
表示将`hello1`这个字符串重定向至1.txt文件里面(就是把`hello1`这个字符串“放进”文件里面;)
```
echo "hello2" >> 1.txt
```
表示将`hello2`这个字符串追加(另起一行追加)重定向至1.txt文件里面，如果不用`>>`这个符号,`hello2`将替换`hello1`
