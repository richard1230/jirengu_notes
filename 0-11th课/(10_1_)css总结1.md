1.如何做横向布局（float + clearfix）:
1).给所有子元素添加 style="float: left;" （让字体横向）

2).所有子元素的爸爸都要添加clearfix类;(修复头像的bug)
```
.clearfix::after{ content: ''; display:block; clear:both;
}
```
